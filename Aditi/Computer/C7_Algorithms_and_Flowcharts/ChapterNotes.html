Algorithms and Flowcharts

Concept

An algorithm is a step-by-step method to solve a problem. A flowchart is a diagrammatic representation of an algorithm using symbols.

1. Characteristics of a Good Algorithm

  • It should be clear and unambiguous.
  • It should end after a finite number of steps.
  • It should be easy to understand and follow.
  • It should produce the correct result.

2. Flowchart Symbols

  • Oval: Start/End
  • Parallelogram: Input/Output
  • Rectangle: Process
  • Diamond: Decision
  • Arrow: Flow line
Fact

Flowcharts help us visualize logic before writing a program.

3. Uses of Algorithms and Flowcharts

They are used in programming, mathematics, and everyday problem solving to plan a solution before execution.

Important

A good algorithm should be simple, precise, and finite.
