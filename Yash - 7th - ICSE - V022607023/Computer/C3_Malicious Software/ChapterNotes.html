Malicious Software

Concept

The term Malware is a portmanteau of two words: Malicious and Software. It refers to any harmful program or piece of code designed to enter a computer without the user's knowledge to perform destructive actions, steal information, or cause annoyance.

1. Types of Computer Viruses

A computer virus specifically replicates itself and infects files. Viruses can be categorized based on the types of files they infect:

Boot Sector Virus

Infects the boot sector (the section of the hard disk used for loading the OS). It loads when the computer boots up. Examples: Elk Cloner, Brain, Disk Killer.

File Infector Virus

Infects program files or executable files (files with an .exe extension). Examples: Cascade, Sunday, Invader.

Macro Virus

Infects files created in applications that use macros (instructions that run automatically), such as Microsoft Word or Excel. Examples: Nuclear, DMV.

Polymorphic Virus

Changes its code every time it infects a new file, making it very difficult for antivirus programs to detect. Examples: Elkern, Marburg.

2. Other Types of Malware

Malware includes several other threats beyond standard viruses:

Trojan Horse

A program that appears legitimate and safe but damages the computer once executed. It does not replicate itself but spreads when copied. It can break passwords or transfer data unauthorizedly. Examples: Acid Rain, Vundo.

Fact

The term Trojan Horse originates from ancient Greek mythology, where Greek warriors hid inside a giant wooden horse given as a gift to enter the city of Troy.

Worms & Spyware

  • Worm: A standalone program that copies itself from one computer to another over a network without needing a host file. Examples: Code Red, Slammer.
  • Spyware: Collects information about your internet surfing habits secretly and uses resources, slowing down the system. Example: Look2Me.

Sweeper & Adware

  • Sweeper: Difficult to detect and re-installs itself even after removal.
  • Adware: Stands for "Advertising Supported Software". It continuously displays banners and pop-up ads on the infected computer.

3. Antivirus Software & Prevention

Antivirus Software is a set of programs used to detect and remove malware from a computer. Examples include free options like AVG and Avira, and paid options like McAfee and Norton.

How Antivirus Works

When an antivirus detects a threat during a system scan, it takes one of three main actions:

Important
  • Quarantining: Placing the infected file in a secure, isolated location so it can no longer harm the system.
  • Repairing: Removing the malicious code and restoring the file to its original safe form.
  • Deleting: Completely removing the file if it is fully corrupted or highly dangerous.
