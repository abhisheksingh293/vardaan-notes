20.
Simplify the following decimal fraction purely by converting to rational fractions or using algebraic identities:

$\displaystyle \frac{0.1 \times 0.1 \times 0.1 - 0.01 \times 0.01 \times 0.01}{0.1 \times 0.1 + 0.1 \times 0.01 + 0.01 \times 0.01}$
Solution Key (For Instructors & Self-Evaluation)
Ans 1. The series is $(-1) + (+1) + (-1) + (+1) \dots$ for $1000$ terms. They form $500$ pairs of $(-1 + 1)$.
Result = $0$.
Ans 2. Take out the common factor: $-345 \times 999 - 345 \times 1 \implies -345 \times (999 + 1) = -345 \times 1000$.
Result = $-345,000$.
Ans 3. Inner parenthesis: $(-5+3) = -2$.
Inner bracket: $-[ -(-2) - 4 ] = -[2 - 4] = -[-2] = 2$.
Curly braces: $-\{ 2 - 6 \} = -\{-4\} = 4$.
Multiply by $(-2)^3$: $4 \times (-8)$.
Result = $-32$.
Ans 4. We need $a \times b \times c = -30$ and $a + b + c = 0$. Factors of $30$ are $1, 2, 3, 5, 6, 10, 15, 30$.
To get a sum of zero, one number must be negative and the other two positive (or vice versa). Let's test $2, 3,$ and $-5$.
Sum: $2 + 3 + (-5) = 0$. Product: $2 \times 3 \times (-5) = -30$.
Result = The integers are $2$, $3$, and $-5$.
Ans 5. Total distance to cover = $15 \text{ m} - (-185 \text{ m}) = 200 \text{ m}$.
Moving time required = $\frac{200}{8} = 25 \text{ minutes}$.
Since it halts every $5 \text{ mins}$ of moving, it will halt after $5, 10, 15,$ and $20$ minutes ($4 \text{ halts}$ in total before reaching the $25 \text{ min}$ mark).
Total halt time = $4 \text{ halts} \times 2 \text{ mins/halt} = 8 \text{ minutes}$.
Total time = $25 + 8$.
Result = $33 \text{ minutes}$.
Ans 6. The sequence from $-2026$ to $+2026$ inevitably includes the integer $0$. Any number multiplied by $0$ is $0$.
Result = $0$.
Ans 7. Bottommost fraction: $1 + \frac{1}{3} = \frac{4}{3}$.
Next layer up: $1 \div \frac{4}{3} = \frac{3}{4}$. Then $1 + \frac{3}{4} = \frac{7}{4}$.
Next layer up: $1 \div \frac{7}{4} = \frac{4}{7}$. Then $1 + \frac{4}{7} = \frac{11}{7}$.
Result = $1 \frac{4}{7}$ (or $\frac{11}{7}$).
Ans 8. Vinculum: $\frac{1}{4} - \frac{1}{6} = \frac{3-2}{12} = \frac{1}{12}$.
Parentheses: $2 \frac{1}{2} - \frac{1}{12} = \frac{5}{2} - \frac{1}{12} = \frac{30-1}{12} = \frac{29}{12}$.
Multiply by $\frac{1}{2}$: $\frac{1}{2} \times \frac{29}{12} = \frac{29}{24}$.
Curly Braces: $1 \frac{1}{4} - \frac{29}{24} = \frac{5}{4} - \frac{29}{24} = \frac{30-29}{24} = \frac{1}{24}$.
Division: $3 \frac{1}{4} \div \frac{1}{24} = \frac{13}{4} \times 24 = 13 \times 6 = 78$.
Final Multiplication: $78 \times \frac{4}{13} = 6 \times 4 = 24$.
Result = $24$.
Ans 9. Let capacity be $X$. Equation: $\frac{3}{5}X - 40 = \frac{1}{3}X$.
$\frac{3}{5}X - \frac{1}{3}X = 40 \implies \frac{9X - 5X}{15} = 40 \implies \frac{4X}{15} = 40$.
$X = 40 \times \frac{15}{4} = 10 \times 15 = 150$.
Result = $150 \text{ litres}$.
Ans 10. LCM of denominators $(12, 18, 8, 24)$ is $72$. Convert to equivalent fractions:
$-\frac{42}{72}, \ -\frac{44}{72}, \ -\frac{45}{72}, \ -\frac{51}{72}$.
Since they are negative, a smaller numerator magnitude means a larger value. Therefore, $-\frac{42}{72} > -\frac{44}{72} > -\frac{45}{72} > -\frac{51}{72}$.
Result = $-\frac{7}{12}, \ -\frac{11}{18}, \ -\frac{5}{8}, \ -\frac{17}{24}$.
Ans 11. Let fractions be $x$ and $y$. $x \cdot y = \frac{10}{3}$ and $\frac{x}{y} = \frac{6}{5}$.
Multiply both equations: $(xy) \times \left(\frac{x}{y}\right) = \frac{10}{3} \times \frac{6}{5} \implies x^2 = \frac{60}{15} = 4 \implies x = 2$.
If $x = 2$, then $2y = \frac{10}{3} \implies y = \frac{5}{3}$.
Sum $= x + y = 2 + \frac{5}{3} = \frac{11}{3}$.
Result = $3 \frac{2}{3}$.
Ans 12. Let total be $X$. Wife gets $\frac{1}{3}X$, remaining is $\frac{2}{3}X$.
Son gets $\frac{1}{4} \text{ of } \frac{2}{3}X = \frac{1}{6}X$. New remaining $= \frac{2}{3}X - \frac{1}{6}X = \frac{1}{2}X$.
Daughter gets $\frac{2}{5} \text{ of } \frac{1}{2}X = \frac{1}{5}X$. Final remaining $= \frac{1}{2}X - \frac{1}{5}X = \frac{3}{10}X$.
$\frac{3}{10}X = 45000 \implies X = 45000 \times \frac{10}{3}$.
Result = ₹ $1,50,000$.
Ans 13. Let fractions be $x, y$. $(x+y) = \frac{45}{8}$ and $(x-y) = \frac{3}{8}$.
Adding both equations: $2x = \frac{48}{8} = 6 \implies x = 3$.
Substitute $x$: $3 + y = \frac{45}{8} \implies y = \frac{45}{8} - \frac{24}{8} = \frac{21}{8}$.
Quotient $\frac{x}{y} = 3 \div \frac{21}{8} = 3 \times \frac{8}{21} = \frac{8}{7}$.
Result = $1 \frac{1}{7}$.
Ans 14. First calculate $\left(\frac{1}{2} \ast \frac{1}{3}\right)$: $\frac{(\frac{1}{2} \times \frac{1}{3})}{(\frac{1}{2} + \frac{1}{3})} = \frac{1/6}{5/6} = \frac{1}{5}$.
Now compute $\frac{1}{5} \ast \frac{1}{4}$: $\frac{(\frac{1}{5} \times \frac{1}{4})}{(\frac{1}{5} + \frac{1}{4})} = \frac{1/20}{9/20} = \frac{1}{9}$.
Result = $\frac{1}{9}$.
Ans 15. Notice the fraction property: $\frac{1}{n(n+1)} = \frac{1}{n} - \frac{1}{n+1}$.
Series expands to: $\left(\frac{1}{1} - \frac{1}{2}\right) + \left(\frac{1}{2} - \frac{1}{3}\right) + \dots + \left(\frac{1}{99} - \frac{1}{100}\right)$.
All intermediate terms cancel out (telescoping series). Leaves $1 - \frac{1}{100}$.
Result = $\frac{99}{100}$.
Ans 16. Since sum is $0$ and $a If $b \ge 0$, then $|a| = -a$, $|b| = b$, $|c| = c$. Equation becomes $-a + b + c = 14 \implies (b+c) + b + c = 14 \implies 2(b+c) = 14 \implies b+c=7$. Valid pairs $(b,c)$ where $b If $b < 0$, then $|a| = -a$, $|b| = -b$, $|c| = c$. Equation: $-a - b + c = 14$. Since $a+b=-c \implies -(-c) + c = 14 \implies 2c=14 \implies c=7$.
Result = The possible values for $c$ are $4, 5, 6,$ or $7$.
Ans 17. Net climb per day $= 5.5 - 2.25 = 3.25 \text{ m/day}$.
Snail reaches the top on the day its starting height $+$ daytime climb $\ge 30 \text{ m}$. Starting height needed before final leap $= 30 - 5.5 = 24.5 \text{ m}$.
Days to reach $24.5 \text{ m}$: $\frac{24.5}{3.25} \approx 7.53 \text{ days}$.
After exactly $7 \text{ days}$, it's at $7 \times 3.25 = 22.75 \text{ m}$. After exactly $8 \text{ days}$, it's at $8 \times 3.25 = 26 \text{ m}$.
On the $9^{\text{th}}$ morning, it starts at $26 \text{ m}$, climbs $5.5 \text{ m}$ to reach $31.5 \text{ m}$, easily crossing the $30 \text{ m}$ mark.
Result = $9^{\text{th}}$ day.
Ans 18. $\frac{1}{1 + 1/x} = \frac{3}{5} \implies 1 + \frac{1}{x} = \frac{5}{3} \implies \frac{1}{x} = \frac{5}{3} - 1 = \frac{2}{3} \implies x = \frac{3}{2}$.
Calculate $2x - \frac{1}{x} = 2\left(\frac{3}{2}\right) - \left(\frac{2}{3}\right) = 3 - \frac{2}{3} = \frac{9-2}{3} = \frac{7}{3}$.
Result = $x = \frac{3}{2}$ and final value is $2 \frac{1}{3}$ (or $\frac{7}{3}$).
Ans 19. Let number be $N$. Intended answer: $N \times \frac{5}{6}$. Wrong answer: $N \div \frac{5}{6} = \frac{6N}{5}$.
Equation: $\frac{6N}{5} - \frac{5N}{6} = 11$.
LCM is $30 \implies \frac{36N - 25N}{30} = 11 \implies \frac{11N}{30} = 11 \implies N = 30$.
Result = The original number was $30$.
Ans 20. Recognize the algebraic identity $a^3 - b^3 = (a - b)(a^2 + ab + b^2)$.
Here $a = 0.1$ and $b = 0.01$. The expression is in the form $\frac{a^3 - b^3}{a^2 + ab + b^2}$.
This simplifies perfectly to $(a - b) \implies 0.1 - 0.01$.
Result = $0.09$ (or $\frac{9}{100}$).
