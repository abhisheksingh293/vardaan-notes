File Management and Data Organisation

Concept

File Management is the process of organising files on a computer. It makes it easier to locate items instead of keeping them in a haphazard manner. It includes copying, moving, saving, and sorting files or folders by name, type, size, and date.

1. Copying and Moving Data

1.1 Copying Data from One Drive to Another

A drive is a location on a computer used to read and store information (e.g., the Hard Disk inside the CPU). Copying duplicates the data so it exists in both the original and new locations.

  • Open File Explorer and locate the file or folder you want to copy.
  • Right-click on it and select the Copy option from the menu.
  • Open the drive where you want to paste the file.
  • Right-click on the blank space in the drive and select the Paste option. The file will be copied.

1.2 Moving Data from One Drive to Another

Moving a file or folder removes it from its original location and shifts it to another destination.

  • Open File Explorer and locate the file or folder.
  • Right-click on it and select the Cut option.
  • Open the drive where you want to place the file.
  • Right-click on the blank space and select the Paste option. The file is now moved.
Fact

Under Windows, the maximum length for a file name (including the drive letter and folder names) is equal to 260 characters.

2. Moving Data Between Storage Devices

A storage device is a hardware tool used to store data (e.g., CD, DVD, Pen Drive). You can transfer data from a computer to a storage device.

2.1 Burning a CD/DVD

The process of transferring data from the memory of a computer to a storage device is called burning.

  • Insert a blank, writable CD/DVD into the CD/DVD drive.
  • The Burn a Disc dialog box appears. Enter a name for the CD/DVD.
  • Select With a CD/DVD player radio button and click Next.
  • The DVD RW Drive opens. Copy and paste the files you want to burn into this drive.
  • Click on the Drive Tools tab under the Manage contextual tab, and click the Finish burning button.
  • The Burn to Disc dialog box opens. Click Next to start the burning process.

2.2 Copying Data from a Pen Drive

  • Insert the pen drive into your computer system's USB port.
  • The USB Drive window appears (or locate it in File Explorer).
  • Right-click the file you want to copy and select Copy.
  • Open the location on your computer where you want to save it, right-click, and select Paste.
Important

Safe Removal: To safely remove a pen drive, click the arrow in the notification area of the taskbar, click the pen drive icon, and select Eject [pen drive name]. Wait for the "Safe To Remove Hardware" message before pulling it out.

3. Working with Multiple Applications

Windows 10 allows you to open two or more applications at the same time and view them side by side.

  • Open the applications you want to use (e.g., a web browser and a music player).
  • Right-click on the empty space on the taskbar.
  • Select the Show windows side by side option from the menu. Both application windows will appear on the screen together.
  • To Undo: Right-click on the taskbar again and select Undo Show all windows side by side.

4. Searching Files Using Wildcard Characters

If you do not know the exact name or location of a file, you can search for it using wildcards.

Concept

A wildcard is a special character used as a substitute for one or more characters when searching for a file on a computer.

4.1 The Asterisk (*) Wildcard

The * character represents one or more unknown characters.

  • Example 1: Searching for emp* will find anything beginning with "emp" (e.g., employee, empire, employ).
  • Example 2: Searching for class.* will locate all files named "class" regardless of their extension (e.g., class.docx, class.pdf, class.xlsx).

4.2 The Question Mark (?) Wildcard

The ? character represents only one unknown character.

  • Example 1: Searching for doll?.docx will display files like doll1.docx or dolls.docx.
  • Example 2: Searching for c??e.docx will display results like came.docx, case.docx, or code.docx.

5. Different File Formats

According to the type of data a file holds (text, image, audio, video), it is saved in a specific file format. The operating system recognises the file type through its file extension (the letters after the dot in a file name).

Format Name Full Form Use & Extension
JPEG Joint Photographic Experts Group Used to store pictures/images. Extension: .jpg or .jpeg
MP3 Moving Pictures Experts Group (Audio Layer 3) Used to store audio and music files. Extension: .mp3
MP4 MPEG-4 Video File A multimedia format used to store video, audio, and subtitles. Extension: .mp4
DOCX Document (Extended) Default format used in Microsoft Word to store formatted text and images. Extension: .docx
XLSX Excel Spreadsheet (Extended) Default format of Microsoft Excel, storing data in tables of rows and columns. Extension: .xlsx
