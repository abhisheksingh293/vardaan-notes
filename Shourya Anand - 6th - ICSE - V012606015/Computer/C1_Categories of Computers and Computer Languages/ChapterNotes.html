Categories of Computers and Computer Languages

Introduction

Just like you need different equipment to play different sports (a tennis racquet for tennis, a bat for cricket), computers are designed to carry out specific tasks. Over the years, computers have changed drastically in size, speed, storage capacity, cost, and appearance.

1. Types of Computers

Computers are divided into various categories based on their size, speed, processing power, and cost.

1.1 Microcomputers

  • These computers work on a single microprocessor (a miniature electronic chip containing all necessary CPU components).
  • Designed to be used by a single person.
  • Examples: Desktop computers, Personal Computers (PCs) used in offices, schools, and homes. Dell, Apple, HP, and IBM are some manufacturers.

1.2 Minicomputers

  • Have more processing power and memory than microcomputers.
  • Can support hundreds of users simultaneously.
  • A large number of input and output devices can be attached.
  • Examples: CDC 1700 and HP 3000.

1.3 Mainframe Computers

  • Very large and expensive computers with massive memory and data storage.
  • Can carry out billions of instructions per second for many people at the same time.
  • Mostly used in big hospitals, airlines, and banks.
  • Examples: IBM Z series.

1.4 Supercomputers

  • The most powerful computers in terms of processing speed and memory.
  • Extremely expensive and capable of executing billions of instructions per second.
  • Used for scientific research, weather forecasting, underground studies, and aircraft designing.
  • Examples: CRAY-1 and IBM's Blue Gene.

1.5 Mobile Computers

The smallest computers designed to be carried around by the user.

  • Netbook: A low-power notebook computer smaller than a laptop. Usually lacks a DVD drive (e.g., ASUS Eee PC).
  • E-book reader: A small handheld device used to read e-books and digital media (e.g., Amazon Kindle).
  • Tablets: Portable computers with a touchscreen interface, larger than a smartphone but smaller than a netbook (e.g., Apple iPad, Microsoft Surface).
  • Smartphones: A combination of a mobile phone and a computer system.
  • PMP (Portable Media Player): Stores and plays audio, images, and videos.

1.6 Game Consoles & Embedded Computers

  • Game Consoles: Devices used for playing computer games on a TV screen using a handheld controller (e.g., Microsoft Xbox, Sony PlayStation, Nintendo Wii).
  • Embedded Computers: A computer integrated into another device to perform a specific function (also called a microcontroller). Used in modern TVs, motor vehicles, digital cameras, microwaves, ATMs, and digital watches.

2. Computer Languages

To communicate with a computer, we must use a language that the computer understands. These are divided into four main categories:

2.1 Machine Language (First Generation - 1GL)

  • The most basic computer language, written in binary form (0s and 1s).
  • The only language directly understood by a computer.
  • Executes very quickly.
  • Machine dependent: A program written for one machine will not work on another.
  • Difficult for humans to use as programmers must remember machine codes.

2.2 Assembly Language (Second Generation - 2GL)

  • Uses symbols and abbreviations instead of 0s and 1s, making it easier for human beings to understand and locate errors.
  • Execution of programs is slower compared to machine language.
  • Also machine dependent.
Low-Level Languages

Both First Generation (Machine Language) and Second Generation (Assembly Language) are considered Low-Level Languages (LLL).

2.3 High-Level Languages (Third Generation - 3GL)

  • Developed to run on any type of computer (Machine Independent).
  • Use English words and mathematical operators, making them easy to learn, read, and write.
  • Examples: C, C++, JAVA, FORTRAN, BASIC, COBOL.

2.4 Fourth Generation Languages (4GL)

  • Modern, highly user-friendly high-level languages.
  • Designed to reduce overall time, effort, and cost of software development.
  • Very high speed of execution compared to previous generations.
  • Examples: SQL, Oracle, Ruby.

2.5 Artificial Intelligence Languages (Fifth Generation - 5GL)

  • Used for developing Artificial Intelligence-based programs.
  • Primary goal is to make the computer solve problems without the programmer having to write a step-by-step algorithm.
  • Aims to make computers think and react like human beings.
  • Examples: Mercury, OPS5, PROLOG.

3. Working of a Translator

Since a computer only understands Machine Language (0s and 1s), programs written in High-level or Assembly languages (called Source Code) cannot be run directly. They must be converted into Machine Language (called Machine Code) using a Translator.

Types of Translators

  • Assembler: Translates a program written in assembly language into machine language.
  • Compiler: Translates a program written in a high-level language into machine language in one go.
  • Interpreter: Translates a program written in a high-level language into machine language line by line.
Compiler vs Interpreter
Compiler Interpreter
Translates the entire program in one go. Translates the code line by line.
Errors are reported after the entire program is translated, making debugging more difficult. Stops translation when the first error is encountered, making the process of removing errors easier and quicker.
Programming languages such as C and C++ use compilers. Programming languages such as Python, BASIC, and Ruby use interpreters.
