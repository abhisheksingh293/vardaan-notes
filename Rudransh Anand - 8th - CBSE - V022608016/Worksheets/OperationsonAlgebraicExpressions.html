Answer Key

Detailed Solutions for Rudransh
Solutions: Section A (Basics)
1.
The terms with identical variable factors are like terms. Here, $3x^2y$ and $5x^2y$ are like terms.
2.
The expression $4x^2 - 3y + 2$ has three separate terms. Therefore, it is a Trinomial.
3.
The term with no variable part is the constant term. Here it is $-12$.
4.
The numerical coefficient is $-7$.
5.
The expression $7xy$ consists of only one term. It is a Monomial.
Solutions: Section B (Addition)
6.
$5x + 7x = \mathbf{12x}$
7.
$(3a + 2a) + (-4b + 3b) + (4c - 8c) = \mathbf{5a - b - 4c}$
8.
$8ab - 5ab + 3ab - ab = \mathbf{5ab}$
9.
$(2x^2 + 5x^2) + (-3x + 6x) + (1 - 4) = \mathbf{7x^2 + 3x - 3}$
10.
$(5x - 2x) + (-8y - 4y) + (2z + 3z) = \mathbf{3x - 12y + 5z}$
11.
$(a + 2a + a) + (-b + b + b) + (c - c + c) = \mathbf{4a + b + c}$
12.
Collecting like terms: $(4x^2 + x^2 + 2x^2) + (-7xy - 8xy - 2xy) + (4y^2 + 6y^2 - 5y^2) + (-3 + 5 + 6) = \mathbf{7x^2 - 17xy + 5y^2 + 8}$
13.
$(x^2 + 2x^2 - x^2) + (-y^2 - 3y^2 + y^2) + (-z^2 + 4z^2 - 2z^2) = \mathbf{2x^2 - 3y^2 + z^2}$
14.
$(3p^2q^2 - 10p^2q^2) + (-4pq + 15pq) + (5 + 9) = \mathbf{-7p^2q^2 + 11pq + 14}$
15.
Perimeter $= 2 \times (\text{Length} + \text{Width}) = 2(5x^2 - 3y^2 + x^2 + 2xy) = 2(6x^2 + 2xy - 3y^2) = \mathbf{12x^2 + 4xy - 6y^2}$
Solutions: Section C (Subtraction)
16.
$8a - 3a = \mathbf{5a}$
17.
$2ab - (-5ab) = 2ab + 5ab = \mathbf{7ab}$
18.
$-5a^2b - 3a^2b = \mathbf{-8a^2b}$
19.
$(3a - 4b - c + 6) - (2a - 5b + 2c - 9) = 3a - 4b - c + 6 - 2a + 5b - 2c + 9 = \mathbf{a + b - 3c + 15}$
20.
$(3x^3 - x^2 + 2x - 4) - (x^3 + 3x^2 - 5x + 4) = 3x^3 - x^2 + 2x - 4 - x^3 - 3x^2 + 5x - 4 = \mathbf{2x^3 - 4x^2 + 7x - 8}$
21.
$(3p^2 - 4q^2 - 5r^2 - 6) - (4p^2 + 5q^2 - 6r^2 + 7) = \mathbf{-p^2 - 9q^2 + r^2 - 13}$
22.
Let the term be $X$. Then, $(3a^2 - 6ab - 3b^2 - 1) - X = (4a^2 - 7ab - 4b^2 + 1)$.
$X = (3a^2 - 6ab - 3b^2 - 1) - (4a^2 - 7ab - 4b^2 + 1) = \mathbf{-a^2 + ab + b^2 - 2}$
23.
$(2x^2 + y^2) - (x^2 - y^2) = 2x^2 + y^2 - x^2 + y^2 = \mathbf{x^2 + 2y^2}$
24.
$(4m^2 - 3mn + 8) - (-m^2 + 5mn) = 4m^2 - 3mn + 8 + m^2 - 5mn = \mathbf{5m^2 - 8mn + 8}$
25.
Sum $= (x + y - 3z) + (-x + y + z) = 2y - 2z$.
Now, subtract: $(2y - 2z) - (x - y + z) = 2y - 2z - x + y - z = \mathbf{-x + 3y - 3z}$
Solutions: Section D (Multiplication)
26.
$(6 \times -3)(x \times x^2)(y \times y^3) = \mathbf{-18x^3y^4}$
27.
$(7 \times -4 \times -5)(a \times a^2 \times a)(b^2 \times b \times b)(c) = 140(a^4)(b^4)(c) = \mathbf{140a^4b^4c}$
28.
$5a^2b^2(3a^2) - 5a^2b^2(4ab) + 5a^2b^2(6b^2) = \mathbf{15a^4b^2 - 20a^3b^3 + 30a^2b^4}$
29.
$-3x^2y(4x^2y) - (-3x^2y)(3xy^2) + (-3x^2y)(4x) - (-3x^2y)(5y) = \mathbf{-12x^4y^2 + 9x^3y^3 - 12x^3y + 15x^2y^2}$
30.
Using distributive law: $2x(3x) + 2x(5y) = \mathbf{6x^2 + 10xy}$
31.
$5x(3x + 4) + 7(3x + 4) = 15x^2 + 20x + 21x + 28 = \mathbf{15x^2 + 41x + 28}$
32.
$2x(4x - 3) + 5(4x - 3) = 8x^2 - 6x + 20x - 15 = \mathbf{8x^2 + 14x - 15}$
33.
$x^2(x + 2y) - y^2(x + 2y) = x^3 + 2x^2y - xy^2 - 2y^3 = \mathbf{x^3 + 2x^2y - xy^2 - 2y^3}$
34.
$x^4(x^2 - y^2) + y^4(x^2 - y^2) = \mathbf{x^6 - x^4y^2 + x^2y^4 - y^6}$
35.
$3x(3x^2 + 5x - 9) - 5(3x^2 + 5x - 9) = 9x^3 + 15x^2 - 27x - 15x^2 - 25x + 45 = \mathbf{9x^3 - 52x + 45}$
Solutions: Section E (Identities)
36.
Using $(a+b)^2$: $(x)^2 + 2(x)(6) + (6)^2 = \mathbf{x^2 + 12x + 36}$
37.
$(4x)^2 + 2(4x)(5y) + (5y)^2 = \mathbf{16x^2 + 40xy + 25y^2}$
38.
Using $(a-b)^2$: $(2x)^2 - 2(2x)(3y) + (3y)^2 = \mathbf{4x^2 - 12xy + 9y^2}$
39.
$(8a)^2 + 2(8a)(3b) + (3b)^2 = \mathbf{64a^2 + 48ab + 9b^2}$
40.
$(7x)^2 + 2(7x)(2y) + (2y)^2 = \mathbf{49x^2 + 28xy + 4y^2}$
41.
$(x^2y)^2 - 2(x^2y)(yz^2) + (yz^2)^2 = \mathbf{x^4y^2 - 2x^2y^2z^2 + y^2z^4}$
42.
Using $(a+b)(a-b) = a^2 - b^2$: $(x)^2 - (3)^2 = \mathbf{x^2 - 9}$
43.
$(2x)^2 - (5)^2 = \mathbf{4x^2 - 25}$
44.
$(50+4)^2 = 50^2 + 2(50)(4) + 4^2 = 2500 + 400 + 16 = \mathbf{2916}$
45.
$(80+2)^2 = 80^2 + 2(80)(2) + 2^2 = 6400 + 320 + 4 = \mathbf{6724}$
46.
$(100+3)^2 = 100^2 + 2(100)(3) + 3^2 = 10000 + 600 + 9 = \mathbf{10609}$
47.
$(70-1)^2 = 70^2 - 2(70)(1) + 1^2 = 4900 - 140 + 1 = \mathbf{4761}$
48.
$(82-18)(82+18) = (64)(100) = \mathbf{6400}$
49.
$(128-72)(128+72) = (56)(200) = \mathbf{11200}$
50.
Numerator: $(198-102)(198+102) = 96 \times 300$.
Expression becomes: $\frac{96 \times 300}{96} = \mathbf{300}$
51.
Notice that $9x^2 + 24x + 16 = (3x)^2 + 2(3x)(4) + (4)^2 = (3x+4)^2$.
Substitute $x=12$: $(3(12)+4)^2 = (36+4)^2 = 40^2 = \mathbf{1600}$
52.
Notice that $64x^2 + 144xy + 81y^2 = (8x)^2 + 2(8x)(9y) + (9y)^2 = (8x+9y)^2$.
Substitute values: $(8(11) + 9(\frac{4}{3}))^2 = (88 + 12)^2 = 100^2 = \mathbf{10000}$
53.
Using $(a-b)^2$: $(\frac{3}{4}x)^2 - 2(\frac{3}{4}x)(\frac{5}{6}y) + (\frac{5}{6}y)^2 = \mathbf{\frac{9}{16}x^2 - \frac{5}{4}xy + \frac{25}{36}y^2}$
54.
$(200 - 3)(200 + 3) = 200^2 - 3^2 = 40000 - 9 = \mathbf{39991}$
55.
Squaring both sides: $(x + \frac{1}{x})^2 = 4^2 \Rightarrow x^2 + 2(x)(\frac{1}{x}) + \frac{1}{x^2} = 16 \Rightarrow x^2 + 2 + \frac{1}{x^2} = 16 \Rightarrow x^2 + \frac{1}{x^2} = 16 - 2 = \mathbf{14}$
