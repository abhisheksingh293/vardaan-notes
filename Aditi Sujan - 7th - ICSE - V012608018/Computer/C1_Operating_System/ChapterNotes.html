Chapter 1: Operating System

Concept

An Operating System (OS) is a collection of system software that acts as an interface between the user and the computer hardware. It controls the execution of all kinds of programs and manages computer hardware resources.

1. Functions of an Operating System

The Operating System is the most vital component of a computer system. Some of its important functions include:

  • Memory Management: Allocates and deallocates memory spaces to various programs.
  • Processor Management: Decides which process gets the processor when and for how much time.
  • Device Management: Keeps track of all hardware devices connected to the computer.
  • File Management: Keeps track of information, location, uses, status, etc., of all files.
  • Security: Protects data and programs from unauthorized access.
  • Control over system performance: Monitors overall health and speed of the computer.
  • Job accounting: Keeps track of time and resources used by various jobs and users.
  • Error detecting aids: Produces dumps, traces, and error messages.
  • Coordination: Manages communication between other software and users.

2. Types of Operating Systems Based on Interface

Operating systems can be broadly classified into two types based on how a user interacts with them:

I. Character User Interface (CUI)

A CUI operating system is text-based. A user has to interact by typing commands on the keyboard to perform specific tasks. Everything is done via command lines.
Examples: DOS (Disk Operating System), UNIX.

II. Graphical User Interface (GUI)

A GUI operating system is a highly visual, mouse-based interface. A user performs tasks by clicking on icons or menus without needing to type code commands or programming languages. It is very user-friendly.
Examples: Windows, Linux.

Important

Differences between GUI and CUI:
• In GUI, you use graphics and pointing devices (like a mouse); in CUI, you type commands via a keyboard.
• GUI allows multiple tasks to run simultaneously (multitasking); CUI generally handles only one task at a time.
• GUI is very user-friendly, whereas CUI requires you to memorize commands.

Features of a GUI

A graphical user interface usually contains these six important features:

  • Pointer: A symbol on the screen that moves as you move your mouse to select commands.
  • Pointing Device: The hardware (like a mouse) that moves the pointer.
  • Icons: Small pictures that represent files, folders, or computer programs.
  • Desktop: The main display area of the screen where you view and organize programs.
  • Windows: Different framed areas on the screen that display different running programs.
  • Menus: Lists that allow users to select and run various commands.

3. Stand-Alone vs. Network Operating System

  • Stand-Alone Operating System: A complete operating system that works independently on a single desktop or notebook computer.
  • Network Operating System (NOS): An operating system designed to support a network (a collection of computers connected via cables or wireless media). In this setup, a Server controls access to the hardware/software, and other computers called Clients rely on the server for resources.

4. Types of Operating Systems Based on Applications

Operating systems are also categorized by the type of computers they control and the tasks they support:

  • Single-user OS: Can support only one user at a time (e.g., Windows 95, MS DOS).
  • Multi-user OS: Allows many different users to use the computer's resources simultaneously (e.g., UNIX, Windows 2000).
  • Multiprocessing OS: Supports more than one processor running at the same time, allocating tasks between them.
  • Multi-tasking OS: Can perform more than one task at a single time using a common CPU (e.g., Windows XP).
  • Real-time operating system (RTOS): Used to control scientific instruments, machinery, and industrial systems. It prioritizes processes to minimize execution time.

5. Popular Operating Systems in the World

Microsoft Windows

Created in the mid-1980s, it is the most popular operating system globally (used by over 80% of users). Recent versions include Windows 10, Windows 8, and Windows 7.

Mac OS X

Created by Apple, it comes pre-loaded on all new Macintosh computers. Specific versions include El Capitan, Yosemite, and Mountain Lion. It is known for its distinct look and feel but is generally more expensive.

Linux

Linux is a family of open-source operating systems. This means unlike Windows (which is proprietary and owned by Microsoft), Linux can be modified and distributed freely by anyone around the world.

Fact

India's very first version of an Open Source Operating System is called BOSS (Bharat Operating System Solutions), developed by C-DAC.

Operating Systems for Mobile Devices

Smartphones, tablets, and MP3 players use operating systems designed specifically for mobile hardware. Examples include Apple iOS and Google Android. While they are not as fully featured as desktop operating systems, they are perfectly optimized for watching movies, browsing the web, making calls, and playing games.
