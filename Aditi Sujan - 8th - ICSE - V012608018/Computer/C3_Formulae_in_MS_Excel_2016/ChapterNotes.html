Formulae in MS Excel 2016

Concept

A formula in Excel is an expression that performs calculation using numbers, cell references, operators, and functions.

1. Basic Formula Structure

All Excel formulas begin with the equals sign =. A formula may contain constants, cell addresses, arithmetic operators, and built-in functions.

  • Example: =A1+B1
  • Example: =SUM(A1:A5)

2. Operators in Excel

  • + Addition
  • - Subtraction
  • * Multiplication
  • / Division
  • ^ Power

3. Cell References

Cell references tell Excel where the data is stored. A relative reference changes when copied, while an absolute reference remains fixed using dollar signs.

Fact

$A$1 is an absolute reference, while A1 is a relative reference.

4. Common Functions

  • SUM() adds values.
  • AVERAGE() finds mean.
  • MAX() returns highest value.
  • MIN() returns lowest value.
  • COUNT() counts numeric entries.

5. Writing Formulas Correctly

Formulas should use proper parentheses and correct order of operations. Use brackets to group calculations when needed.

Important

Excel follows the mathematical order of operations, so multiplication and division are done before addition and subtraction.
