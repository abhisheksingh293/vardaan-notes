Creating Charts in MS Excel 2016

Concept

Charts help present numerical data in a visual form so that comparisons and trends are easy to understand.

1. Why Use Charts

  • To compare values quickly.
  • To show trends over time.
  • To make reports attractive and clear.

2. Common Chart Types

  • Column Chart: Displays vertical bars.
  • Bar Chart: Displays horizontal bars.
  • Line Chart: Shows trends over a period.
  • Pie Chart: Shows parts of a whole.
Fact

A chart is linked to worksheet data, so if the data changes, the chart can update too.

3. Steps to Create a Chart

  1. Select the data range.
  2. Insert the required chart type.
  3. Choose a layout and chart style.
  4. Add title, labels, and legend.

4. Formatting a Chart

You can change chart colours, titles, axes, and data labels to make the chart more useful and easier to read.

Important

Always choose the chart type that best matches the data you want to show.
