Chapter 6: Creating Tables & Marquee in HTML

Concept

Tables in HTML are used to arrange data like text, images, or links into rows and columns. This makes the data easy to read and understand. An HTML Marquee is used to create moving (scrolling) text or images on a webpage.

1. Creating an HTML Table

To create a table in HTML, we use specific tags for the table, rows, and data cells:

  • <table>: This tag is used to start and define the entire table.
  • <tr> (Table Row): This tag is used to divide the table into horizontal rows.
  • <td> (Table Data): This tag is used to create individual cells inside a row to hold data. The number of <td> tags determines the number of columns.
  • <th> (Table Heading): This tag is used in place of <td> when you want to create a heading for a column or row. The text inside <th> automatically becomes bold and centered.

2. Table Attributes & Formatting

You can make your tables look beautiful by adding special attributes inside the <table> tag.

The Border Attribute

If you don't use the border attribute, your table will display without any visible lines (borders). To add lines, use the border attribute.

Fact

Example: <table border="1"> tells the browser to draw a border that is 1 pixel thick around the table and its cells.

Formatting Attributes

  • bgcolor: Sets the background color for the entire table or a single cell (e.g., bgcolor="orange").
  • background: Sets a background image for the table or a single cell.
  • bordercolor: Changes the color of the table's border (e.g., bordercolor="red").
  • width & height: Can be set in pixels (like 300) or percentages (like 50%). Using 50% means the table takes up half the width of the webpage.
  • align: Used to place the table in the left, right, or center of the page.

3. Table Caption, Cellpadding, and Cellspacing

Table Caption

The <caption> tag is used to give a title to your table. The caption appears at the very top of the table.

Cellpadding and Cellspacing

These two attributes help you adjust the blank white space inside and around the cells of your table.

  • cellspacing: Defines the width of the empty space between two different cells (or the thickness of the border space).
  • cellpadding: Represents the distance between the cell's border and the text/content inside that cell.
Important

Quick Trick to Remember:
• Cellspacing = Space OUTSIDE the cell (between cells).
• Cellpadding = Space INSIDE the cell (padding around the text).

4. Spanning Rows and Columns

Sometimes you need a single cell to stretch across multiple columns or multiple rows. HTML allows this using two special attributes inside the <td> or <th> tags:

  • colspan: Makes a cell span across more than one column. (Example: <th colspan="3">Marks</th> makes the "Marks" heading cover 3 columns).
  • rowspan: Makes a cell span vertically across more than one row.

5. HTML Marquees

An HTML marquee is a scrolling piece of text or image displayed horizontally or vertically on a webpage. This is created using the <marquee> tag.

Marquee Attributes

  • direction: Specifies which way the text should move. Values can be up, down, left, or right.
  • behavior: Specifies how the marquee scrolls.
    • scroll: Scrolls continuously off the screen and comes back.
    • slide: Slides in and stops once it hits the edge.
    • alternate: Bounces back and forth between the edges.
  • bgcolor: Specifies the background color behind the moving text.

Using Images in a Marquee

You can also place images inside a marquee to make the picture move across the screen. You simply put the <img> tag between the opening and closing marquee tags.

Example:
<marquee direction="left"> <img src="bird.gif"> </marquee>
