Elaboration on MS Excel 2016

Concept

Excel becomes powerful when we use references, formatting, sorting, filtering, and data tools together in a worksheet.

1. Cell References

Cell references identify where a value is stored. Relative references change when copied, absolute references stay fixed, and mixed references combine both styles.

2. Functions in Worksheets

Functions make repeated calculations easy. You can use them in marksheets, bills, totals, and comparisons.

  • SUM for total
  • AVERAGE for mean
  • MAX and MIN for extreme values

3. Organising Data

Excel can sort data in ascending or descending order. Filtering helps display only the rows that match a condition.

Fact

Sort rearranges all data, while Filter hides unwanted records temporarily.

4. Formatting Cells

  • Change font, colour, and alignment.
  • Apply borders and fill colour.
  • Use number formats like currency and percentage.

5. Validation and Protection

Data validation helps enter correct values, and worksheet protection prevents unwanted changes.

Important

Well-formatted worksheets are easier to read, update, and present to others.
