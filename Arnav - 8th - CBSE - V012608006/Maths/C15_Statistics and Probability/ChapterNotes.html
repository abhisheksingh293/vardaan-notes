Statistics and Probability

1. Raw Data and Presentation

Statistics is the science of collecting, classifying, summarizing, analyzing, and interpreting numerical facts. In various fields, we need information in the form of numerical figures.

Concept
  • Observation: Each individual entry or numerical figure in a given list is called an observation.
  • Raw Data: The collection of all these initial, unorganized observations is called data or raw data.

Presentation of Data

After collection, an investigator tries to arrange data so its salient features (lowest score, highest score, etc.) can be studied easily. The best way is to arrange the data in ascending or descending order. This arrangement is called the presentation of data.

Range of Data

The difference between the highest and the lowest values of the observations is called the range of the data.

$Range = \text{Highest Value} - \text{Lowest Value}$

2. Frequency Distribution Table

If the number of observations is large, finding information by simple counting is tedious. We organize data into a table using Tally Marks.

Fact

Frequency: The number of times a particular observation occurs in a given data is called its frequency.

Tally Marks: To facilitate counting, we record tally marks in bunches of five using bars ( | ). The fifth tally mark is drawn diagonally across the first four ( $\cancel{||||}$ ).

Grouping of Data

When the difference between the greatest and smallest observation is very large, organizing individual frequencies takes too much space. To overcome this, we divide raw data into a convenient number of groups called class intervals (e.g., 0-5, 5-10, 10-15).

  • Limits: In the class 0-5, 0 is the lower limit and 5 is the upper limit. (Note: The upper limit is generally excluded from that specific class interval; e.g., 5 falls in the 5-10 class, not 0-5).
  • Class Size (Width): The difference between the upper limit and the lower limit. Example: $5 - 0 = 5$.
  • Class Mark: The average of the upper and lower limits.

$Class\ Mark = \frac{\text{Lower Limit} + \text{Upper Limit}}{2}$

Practice Problem 1: Class Marks & Intervals
For a class interval of 120-140:
(i) What is the lower limit?
(ii) What is the upper limit?
(iii) Calculate the class size and class mark.

Solution:
(i) Lower limit = 120.
(ii) Upper limit = 140.
(iii) Class size = $140 - 120 = 20$. Class mark = $\frac{120 + 140}{2} = 130$.

3. Graphical Method: Histogram

A histogram is a graphical representation of a grouped frequency distribution. It is depicted in the form of rectangles with class intervals as bases and heights proportional to the corresponding frequencies.

Key characteristic: There is no gap between any two successive rectangles because the class intervals are continuous.

Concept

The Kink ($\approx$): If the first class interval does not start from zero, we draw a 'kink' or zig-zag line on the horizontal axis near the origin. It shows that the full distance from 0 to the starting point is omitted to save space.

4. Circle Graph or Pie Chart

A circle graph or pie chart shows the relationship between a whole and its parts. The circle is divided into different sectors, where the size (angle) of each sector is proportional to the information it represents.

Since the total angle at the center of a circle is $360^{\circ}$, the central angle of a sector is calculated as:

$Central\ Angle = \left( \frac{\text{Value of Component}}{\text{Total Value}} \right) \times 360^{\circ}$

Example: Fruit Sales

Fruit % Sold Fraction Central Angle
Apple 25% $\frac{25}{100} = \frac{1}{4}$ $\frac{1}{4} \times 360^{\circ} = 90^{\circ}$
Mango 50% $\frac{50}{100} = \frac{1}{2}$ $\frac{1}{2} \times 360^{\circ} = 180^{\circ}$
Banana 25% $\frac{25}{100} = \frac{1}{4}$ $\frac{1}{4} \times 360^{\circ} = 90^{\circ}$

5. Probability

In our day-to-day life, we use words like "probably", "likely", or "chances" when there is an element of uncertainty.

  • Experiment: An operation which can produce some well-defined outcome(s).
  • Trial: When we perform an experiment, it is called a trial (e.g., tossing a coin).
  • Event: An outcome of the trial (e.g., getting a "Head").
  • Equally Likely Outcomes: Outcomes of a trial where each outcome has the exact same chance of occurrence (like rolling a fair 6-sided die).
Fact

The Probability Formula:

$Probability\ of\ an\ event\ P(E) = \frac{\text{Number of outcomes that make the event}}{\text{Total number of possible outcomes of the experiment}}$

Examples

1. Tossing a coin: Total outcomes = 2 (Head, Tail).
$P(\text{Getting a Head}) = \frac{1}{2}$.

2. Throwing a die: Total outcomes = 6 (1, 2, 3, 4, 5, 6).
Getting an even number means getting 2, 4, or 6 (3 favorable outcomes).
$P(\text{Getting an even number}) = \frac{3}{6} = \frac{1}{2}$.

Practice Problem 3: Probability Challenges
(a) There are 24 marbles in a box: 10 Red, 6 Green, 5 Yellow, and 3 Blue. If you pick one without looking, what is the probability of getting a yellow marble?
(b) A well-shuffled deck has 52 playing cards. What is the probability of getting an Ace?

Solutions:
(a) Total outcomes = 24. Favorable (Yellow) = 5. Probability = $\frac{5}{24}$.
(b) Total outcomes = 52. Favorable (Aces) = 4. Probability = $\frac{4}{52} = \frac{1}{13}$.
