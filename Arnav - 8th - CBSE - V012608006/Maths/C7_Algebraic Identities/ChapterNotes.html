7. Algebraic Identities

In previous classes, you have learnt multiplication of algebraic expressionsâ€”monomials, binomials and trinomials. Now consider the following statement involving an algebraic expression: $4x + 3y = 10$.

If we substitute $x = 2$, $y = 2$, then $L.H.S. = 8 + 6 = 14$. Thus, $L.H.S. \neq R.H.S.$

If we substitute $x = 1$, $y = 2$, then $L.H.S. = 4 + 6 = 10$. Thus, $L.H.S. = R.H.S.$

Concept

Identity: A statement of equality which holds for all values of the variables is called an identity.

Thus, $4x + 3y = 10$ is an equation, not an identity, because it is only true for certain values of the variables.

In this chapter, you will learn the identities which arise as the product of a binomial $(a + b)$ with itself, $(a - b)$ with itself, and $(a + b)$ with $(a - b)$.

1. Standard Identities

I. Identity-I

Just like the square of a number means multiplying the number with itself, the square of a binomial means multiplying the binomial with itself.

Identity-I

$(a + b)^2 = a^2 + 2ab + b^2$

Proof:
$(a + b)^2 = (a + b)(a + b)$
$= a(a + b) + b(a + b)$
$= a^2 + ab + ba + b^2$
$= a^2 + 2ab + b^2 \quad [\because ab = ba]$

Example 1: Find $(3x + 4y)^2$ using Identity-I.
Solution: $(3x + 4y)^2 = (3x)^2 + 2 \times 3x \times 4y + (4y)^2$
$= 9x^2 + 24xy + 16y^2$

Example 2: Evaluate $42^2$ by using Identity-I.
Solution: $42^2 = (40 + 2)^2$
$= 40^2 + 2 \times 40 \times 2 + 2^2$
$= 1600 + 160 + 4 = 1764$

II. Identity-II

You can get this identity by multiplying binomial $a - b$ with itself.

Identity-II

$(a - b)^2 = a^2 - 2ab + b^2$

Proof:
$(a - b)^2 = (a - b)(a - b)$
$= a(a - b) - b(a - b)$
$= a^2 - ab - ba + b^2$
$= a^2 - 2ab + b^2 \quad [\because ba = ab]$

Note: Identity-II can also be considered as a special case of Identity-I when '$b$' is replaced by '$-b$'.

Example 3: Find $(2x - 3y)^2$ using Identity-II.
Solution: $(2x - 3y)^2 = (2x)^2 - 2 \times 2x \times 3y + (3y)^2$
$= 4x^2 - 12xy + 9y^2$

Example 4: Evaluate $99^2$ using Identity-II.
Solution: $99^2 = (100 - 1)^2 = 100^2 - 2 \times 100 \times 1 + 1^2$
$= 10000 - 200 + 1 = 9801$

III. Identity-III

You can easily multiply the two binomials $(a + b)$ and $(a - b)$ to get this identity.

Identity-III

$(a + b)(a - b) = a^2 - b^2$

Proof:
$(a + b)(a - b) = a(a - b) + b(a - b)$
$= a^2 - ab + ba - b^2$
$= a^2 - b^2$

Example 5: Find the product $(9x + 5y)(9x - 5y)$ by using Identity-III.
Solution: $(9x + 5y)(9x - 5y) = (9x)^2 - (5y)^2$
$= 81x^2 - 25y^2$

Example 6: Simplify $93 \times 87$ by expressing it as a difference of two squares.
Solution: $93 \times 87 = (90 + 3)(90 - 3)$
$= 90^2 - 3^2$
$= 8100 - 9 = 8091$

2. Expansion of the Square of a Trinomial

Let us take a trinomial $(a + b + c)$. Let $a + b = x$. Now, square the trinomial as follows:

$(a + b + c)^2 = (x + c)^2$
$= x^2 + 2xc + c^2$ (Using Identity I)
$= (a + b)^2 + 2(a + b)c + c^2$
$= a^2 + 2ab + b^2 + 2ac + 2bc + c^2$
$= a^2 + b^2 + c^2 + 2ab + 2bc + 2ca$

Important Expansion

$(a + b + c)^2 = a^2 + b^2 + c^2 + 2ab + 2bc + 2ca$

Example 7: Expand $(3x + 2y + 7)^2$
Solution: $(3x + 2y + 7)^2 = (3x)^2 + (2y)^2 + 7^2 + 2 \times 3x \times 2y + 2 \times 2y \times 7 + 2 \times 3x \times 7$
$= 9x^2 + 4y^2 + 49 + 12xy + 28y + 42x$

3. Product of Two Binomials: $(x + a)(x + b)$

Let's multiply two general binomials that share a common first term:

$(x + a)(x + b) = x(x + b) + a(x + b)$
$= x^2 + xb + ax + ab$
$= x^2 + (a + b)x + ab$

Fact File

$(x + a)(x + b) = x^2 + (a + b)x + ab$

From this, we can derive:

  • $(x + a)(x - b) = x^2 + (a - b)x - ab$
  • $(x - a)(x + b) = x^2 + (b - a)x - ab$
  • $(x - a)(x - b) = x^2 - (a + b)x + ab$

Example 8: Find the product $(p + 6)(p - 3)$.
Solution: Using $(x + a)(x - b) = x^2 + (a - b)x - ab$
$(p + 6)(p - 3) = p^2 + (6 - 3)p - (6 \times 3)$
$= p^2 + 3p - 18$

Example 9: Evaluate $103 \times 105$ using the identity.
Solution: $103 \times 105 = (100 + 3)(100 + 5)$
$= 100^2 + (3 + 5)100 + (3 \times 5)$
$= 10000 + 800 + 15 = 10815$

4. Factorisation of Algebraic Expressions

Concept

Factorisation: The process of writing a given algebraic expression as the product of two or more factors is called factorisation.

A. Factorisation by using Identities

We use the reverse of our standard identities to factorise perfect square trinomials or differences of squares.

Example 10: Factorise $y^2 + 4y + 4$
Solution: We can write it as $(y)^2 + 2 \times y \times 2 + (2)^2$
$= (y + 2)^2 = (y + 2)(y + 2)$

Example 11: Factorise $9y^2 - 16$
Solution: $9y^2 - 16 = (3y)^2 - (4)^2$
$= (3y - 4)(3y + 4)$ (Using $a^2 - b^2 = (a - b)(a + b)$)

Example 12: Factorise $x^4 - y^4$
Solution: $x^4 - y^4 = (x^2)^2 - (y^2)^2$
$= (x^2 - y^2)(x^2 + y^2)$
$= (x - y)(x + y)(x^2 + y^2)$ (Applying the identity repeatedly)

B. Factorisation of the Form $x^2 + qx + r$ (Splitting the Middle Term)

To factorise this, we use the identity $x^2 + (a + b)x + ab = (x + a)(x + b)$. We need to find two numbers $a$ and $b$ whose product is $r$ and whose sum is $q$.

Example 13: Factorise $x^2 + 9x + 18$
Solution: We need two numbers whose product is 18 and sum is 9. The factors of 18 are $(1 \times 18), (2 \times 9), (3 \times 6)$. The pair that sums to 9 is 3 and 6.
$x^2 + 9x + 18 = x^2 + 6x + 3x + 18$
$= x(x + 6) + 3(x + 6)$
$= (x + 6)(x + 3)$

Example 14: Factorise $x^2 - 4x - 45$
Solution: Product is $-45$, Sum is $-4$. The numbers are $-9$ and $5$.
$x^2 - 4x - 45 = x^2 - 9x + 5x - 45$
$= x(x - 9) + 5(x - 9) = (x - 9)(x + 5)$
