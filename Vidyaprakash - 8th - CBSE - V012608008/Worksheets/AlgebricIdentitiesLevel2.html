Algebraic Identities - Level 2 (Intermediate)
Especially designed for: Aarav Class: 8 CBSE Subject: Mathematics
Section A: Fractional & Decimal Manipulations
1.
Expand: $\left(\frac{x}{2} + \frac{2}{y}\right)^2$
2.
Evaluate using standard identities: $10.2 \times 9.8$
3.
Find the product: $(2x + 5)(2x - 3)$
4.
Find the value of: $\frac{78 \times 78 - 22 \times 22}{56}$
Section B: Inverse Value Finding
5.
If $x + \frac{1}{x} = 5$, find the value of $x^2 + \frac{1}{x^2}$.
6.
If $x^2 + \frac{1}{x^2} = 27$, find the value of $x - \frac{1}{x}$.
7.
If $a - b = 6$ and $ab = 16$, find $a^2 + b^2$.
Section C: Factorization & Proofs
8.
Factorize using standard identities: $49x^2 - 84xy + 36y^2$
9.
Simplify: $(p^2 - q^2)^2 + 4p^2q^2$
10.
Show that: $(3x + 7)^2 - 84x = (3x - 7)^2$
Answer Key
1.
Ans: $\left(\frac{x}{2}\right)^2 + 2\left(\frac{x}{2}\right)\left(\frac{2}{y}\right) + \left(\frac{2}{y}\right)^2 = \frac{x^2}{4} + \frac{2x}{y} + \frac{4}{y^2}$
2.
Ans: $(10 + 0.2)(10 - 0.2) = 10^2 - 0.2^2 = 100 - 0.04 = 99.96$
3.
Ans: Using $(x+a)(x+b)$: $(2x)^2 + (5 - 3)(2x) + (5)(-3) = 4x^2 + 2(2x) - 15 = 4x^2 + 4x - 15$
4.
Ans: $\frac{78^2 - 22^2}{56} = \frac{(78 - 22)(78 + 22)}{56} = \frac{56 \times 100}{56} = 100$
5.
Ans: $(x + \frac{1}{x})^2 = x^2 + \frac{1}{x^2} + 2 \implies 5^2 = x^2 + \frac{1}{x^2} + 2 \implies x^2 + \frac{1}{x^2} = 25 - 2 = 23$
6.
Ans: $(x - \frac{1}{x})^2 = x^2 + \frac{1}{x^2} - 2 = 27 - 2 = 25 \implies x - \frac{1}{x} = \pm 5$
7.
Ans: $(a-b)^2 = a^2 + b^2 - 2ab \implies 6^2 = a^2 + b^2 - 2(16) \implies 36 = a^2 + b^2 - 32 \implies a^2 + b^2 = 68$
8.
Ans: $49x^2 = (7x)^2$ and $36y^2 = (6y)^2$. Middle term: $-2(7x)(6y) = -84xy$. So, $(7x - 6y)^2$.
9.
Ans: $(p^2 - q^2)^2 + 4p^2q^2 = (p^4 - 2p^2q^2 + q^4) + 4p^2q^2 = p^4 + 2p^2q^2 + q^4 = (p^2 + q^2)^2$
10.
Ans: LHS: $(3x)^2 + 2(3x)(7) + 7^2 - 84x = 9x^2 + 42x + 49 - 84x = 9x^2 - 42x + 49$.
RHS: $(3x - 7)^2 = 9x^2 - 42x + 49$. LHS = RHS. Proved.
