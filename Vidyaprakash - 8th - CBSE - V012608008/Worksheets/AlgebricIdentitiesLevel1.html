Algebraic Identities - Level 1 (Foundation)
Especially designed for: Aarav Class: 8 CBSE Subject: Mathematics
Section A: Direct Application of Identities
1.
Expand using a suitable identity: $(2x + 3y)^2$
2.
Expand using a suitable identity: $(4p - 5q)^2$
3.
Find the product using the identity $(a+b)(a-b) = a^2 - b^2$: $(3m + 4n)(3m - 4n)$
4.
Multiply using the identity $(x+a)(x+b) = x^2 + (a+b)x + ab$: $(x + 4)(x + 7)$
5.
Simplify: $(2a + 3b)^2 - (2a - 3b)^2$
Section B: Numerical Evaluations
6.
Evaluate using standard identities: $103^2$
7.
Evaluate using standard identities: $98^2$
8.
Evaluate using standard identities: $51 \times 49$
Section C: Basic Value Finding
9.
Find the value of $(x+y)^2$ if $x^2+y^2=25$ and $xy=12$.
10.
If $a+b=7$ and $ab=10$, find the value of $a^2+b^2$.
Answer Key
1.
Ans: Using $(a+b)^2 = a^2 + 2ab + b^2$.
$(2x + 3y)^2 = (2x)^2 + 2(2x)(3y) + (3y)^2 = 4x^2 + 12xy + 9y^2$
2.
Ans: Using $(a-b)^2 = a^2 - 2ab + b^2$.
$(4p - 5q)^2 = (4p)^2 - 2(4p)(5q) + (5q)^2 = 16p^2 - 40pq + 25q^2$
3.
Ans: Using $(a+b)(a-b) = a^2 - b^2$.
$(3m)^2 - (4n)^2 = 9m^2 - 16n^2$
4.
Ans: $(x + 4)(x + 7) = x^2 + (4 + 7)x + (4)(7) = x^2 + 11x + 28$
5.
Ans: Expand both: $[(2a)^2 + 2(2a)(3b) + (3b)^2] - [(2a)^2 - 2(2a)(3b) + (3b)^2]$
$= (4a^2 + 12ab + 9b^2) - (4a^2 - 12ab + 9b^2) = 12ab + 12ab = 24ab$.
*(Alternatively, use $x^2-y^2 = (x+y)(x-y)$)*
6.
Ans: $103^2 = (100 + 3)^2 = 100^2 + 2(100)(3) + 3^2 = 10000 + 600 + 9 = 10609$
7.
Ans: $98^2 = (100 - 2)^2 = 100^2 - 2(100)(2) + 2^2 = 10000 - 400 + 4 = 9604$
8.
Ans: $51 \times 49 = (50 + 1)(50 - 1) = 50^2 - 1^2 = 2500 - 1 = 2499$
9.
Ans: $(x+y)^2 = x^2 + y^2 + 2xy = 25 + 2(12) = 25 + 24 = 49$.
10.
Ans: $(a+b)^2 = a^2 + b^2 + 2ab \implies 7^2 = a^2 + b^2 + 2(10) \implies 49 = a^2 + b^2 + 20 \implies a^2 + b^2 = 29$.
