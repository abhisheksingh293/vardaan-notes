Exponents and Powers - Level 3 (Challenger)
Especially designed for: Aarav Class: 8 CBSE Subject: Mathematics
Section A: Higher Order Simplification
1.
Simplify: $\frac{16 \times 2^{n+1} - 4 \times 2^n}{16 \times 2^{n+2} - 2 \times 2^{n+2}}$
2.
If $a = \left(\frac{3}{5}\right)^{-2} \div \left(\frac{7}{5}\right)^0$, find the value of $a^{-3}$.
3.
Simplify: $\frac{x^a}{x^b} \times \frac{x^b}{x^c} \times \frac{x^c}{x^a}$.
4.
Find the value of $\left[ \left( -\frac{1}{3} \right)^{-2} - \left( \frac{1}{2} \right)^{-3} \right] \div \left( \frac{1}{4} \right)^{-2}$
5.
Simplify: $\frac{9^n \times 3^2 \times \left(3^{\frac{-n}{2}}\right)^{-2} - 27^n}{3^{3m} \times 2^3} = \frac{1}{27}$, and express $m$ in terms of $n$.
Section B: Complex Equations (Find $x$)
6.
If $\left(\frac{p}{q}\right) = \left(\frac{2}{3}\right)^2 \times \left(\frac{1}{3}\right)^{-4}$, find the value of $\left(\frac{p}{q}\right)^{-2}$.
7.
Find $x$ if $\left(\frac{2}{3}\right)^x \times \left(\frac{3}{2}\right)^{2x} = \frac{81}{16}$
8.
Solve for $x$: $3^{2x+1} + 3^{2x} = 108$
9.
Find $x$ if $5^{x-2} \times 3^{2x-3} = 135$
10.
By what number should $(-3/2)^{-3}$ be divided so that the quotient is $(9/4)^{-2}$?
Section C: Conceptual & HOTS Problems
11.
A particular star is at a distance of about $8.1 \times 10^{13} \text{ km}$ from Earth. Assume that light travels at $3 \times 10^5 \text{ km/sec}$. Find how long light takes from that star to reach Earth. Express the answer in standard form (in seconds).
12.
If $2^x - 2^{x-1} = 4$, find the value of $x^x$.
13.
Evaluate: $\frac{3^{n+1} - 3^n}{3^{n+2} - 3^n}$
14.
Is $(a^m)^n = a^{m^n}$? Verify with $a=2, m=3, n=2$.
15.
If $1176 = 2^a \times 3^b \times 7^c$, find the values of $a, b,$ and $c$. Hence, compute the value of $2^a \times 3^b \times 7^{-c}$ as a fraction.
Answer Key
1.
Ans: $\frac{2^4 \cdot 2^{n+1} - 2^2 \cdot 2^n}{2^4 \cdot 2^{n+2} - 2^1 \cdot 2^{n+2}} = \frac{2^{n+5} - 2^{n+2}}{2^{n+6} - 2^{n+3}} = \frac{2^{n+2}(2^3 - 1)}{2^{n+3}(2^3 - 1)} = \frac{2^{n+2} \cdot 7}{2^{n+3} \cdot 7} = \frac{1}{2}$
2.
Ans: $a = \left(\frac{5}{3}\right)^2 \div 1 = \frac{25}{9}$. So $a^{-3} = \left(\frac{9}{25}\right)^3 = \frac{729}{15625}$
3.
Ans: $x^{a-b} \times x^{b-c} \times x^{c-a} = x^{a-b+b-c+c-a} = x^0 = 1$
4.
Ans: $[(-3)^2 - 2^3] \div 4^2 = [9 - 8] \div 16 = \frac{1}{16}$
5.
Ans: Numerator: $3^{2n} \cdot 3^2 \cdot 3^n - 3^{3n} = 3^{3n+2} - 3^{3n} = 3^{3n}(3^2 - 1) = 3^{3n} \cdot 8$.
Denom: $3^{3m} \cdot 8$. Ratio: $\frac{3^{3n}}{3^{3m}} = 3^{3n-3m} = 3^{-3}$. $\therefore 3n-3m = -3 \implies m - n = 1 \implies m = n + 1$
6.
Ans: $\frac{p}{q} = \frac{4}{9} \times 3^4 = \frac{4}{9} \times 81 = 36$. $\left(\frac{p}{q}\right)^{-2} = 36^{-2} = \frac{1}{1296}$
7.
Ans: $\left(\frac{3}{2}\right)^{-x} \times \left(\frac{3}{2}\right)^{2x} = \left(\frac{3}{2}\right)^4 \implies \left(\frac{3}{2}\right)^x = \left(\frac{3}{2}\right)^4 \implies x = 4$
8.
Ans: $3^{2x}(3^1 + 1) = 108 \implies 3^{2x} \cdot 4 = 108 \implies 3^{2x} = 27 \implies 3^{2x} = 3^3 \implies 2x = 3 \implies x = \frac{3}{2}$
9.
Ans: $135 = 5^1 \times 3^3$. By comparison, $x-2 = 1 \implies x = 3$. Checking with $2x-3 = 2(3)-3 = 3$. Consistent. $x = 3$.
10.
Ans: Let divisor be $y$. $\left(-\frac{2}{3}\right)^3 \div y = \left(\frac{4}{9}\right)^2 \implies \left(-\frac{8}{27}\right) \times \frac{1}{y} = \frac{16}{81} \implies y = -\frac{8}{27} \times \frac{81}{16} = -\frac{3}{2}$
11.
Ans: Time = $\frac{\text{Distance}}{\text{Speed}} = \frac{8.1 \times 10^{13}}{3 \times 10^5} = 2.7 \times 10^8 \text{ sec}$
12.
Ans: $2^{x-1}(2 - 1) = 4 \implies 2^{x-1} \times 1 = 2^2 \implies x-1 = 2 \implies x = 3$. Then $x^x = 3^3 = 27$
13.
Ans: $\frac{3^n(3^1 - 1)}{3^n(3^2 - 1)} = \frac{2}{8} = \frac{1}{4}$
14.
Ans: LHS: $(2^3)^2 = 8^2 = 64$. RHS: $2^{(3^2)} = 2^9 = 512$. LHS $\neq$ RHS. So, $(a^m)^n \neq a^{m^n}$.
15.
Ans: $1176 = 2^3 \times 3^1 \times 7^2 \implies a=3, b=1, c=2$. Value required = $2^3 \times 3^1 \times 7^{-2} = \frac{8 \times 3}{49} = \frac{24}{49}$
